# Automation Recipes

Ready-to-use prompt templates for common LinkedIn workflows. Copy these into Claude Desktop and customize for your needs.

---

## Content Creation

### Quick Post

```
Post on LinkedIn: [your text here]
```

### Post with Hashtags

```
Create a LinkedIn post about [topic]. Add hashtags: [tag1], [tag2], [tag3]
```

### Post from an Idea

```
I want to post on LinkedIn about [topic]. Write an engaging post with:
- A hook in the first line
- 3 short paragraphs
- A call to action at the end
- 5 relevant hashtags

Show me the draft first, then post it after I approve.
```

### Engagement-Optimized Post

```
Here's my draft LinkedIn post:

[paste your draft]

Rewrite it for maximum engagement:
- Strong hook in the first line (pattern interrupt or surprising stat)
- Short paragraphs (1-2 sentences each)
- Use line breaks for readability
- End with a question to drive comments
- Add 5 relevant hashtags

Show me the rewrite, then post it when I say go.
```

## Scheduling

### Schedule a Single Post

```
Schedule a LinkedIn post for [date] at [time]: [your text]
```

### Batch Schedule (Content Calendar)

```
I have 5 topics for this week. Draft a LinkedIn post for each one and schedule them:

1. Monday 9am: [topic 1]
2. Tuesday 9am: [topic 2]
3. Wednesday 9am: [topic 3]
4. Thursday 9am: [topic 4]
5. Friday 9am: [topic 5]

Show me all drafts first for approval before scheduling.
```

### Weekly Content Calendar

```
Help me plan my LinkedIn content for next week. I want to post about:
- Monday: A professional insight about [industry/field]
- Wednesday: A tip or lesson learned from [recent experience]
- Friday: A personal reflection or weekend thought

Draft all three posts, show them to me, then schedule them for 10am each day.
```

## Advanced Workflows

### Thread-Style Long Content

LinkedIn doesn't have threads, but you can simulate them with long posts:

```
I want to write a long LinkedIn post about [topic]. Structure it as:

1. Hook (1 line that grabs attention)
2. Context (2-3 sentences of background)
3. Main points (numbered list, 3-5 items, 1-2 sentences each)
4. Key takeaway (1 bold statement)
5. Call to action (question or ask)
6. Hashtags (5-7 relevant ones)

Keep it under 3000 characters. Show me the draft.
```

### Repurpose Content Across Formats

```
Here's a talk abstract I submitted to a conference:

[paste abstract]

Turn this into a LinkedIn post announcing that I'm speaking at [conference name] on [date]. Make it personal and excited, not salesy. Add relevant hashtags.
```

### Thought Leadership Series

```
I want to create a 5-part series on LinkedIn about [broad topic].

Plan the series:
- Part 1: [subtopic suggestion]
- Part 2: [subtopic suggestion]
- Part 3: [subtopic suggestion]
- Part 4: [subtopic suggestion]
- Part 5: [subtopic suggestion]

Draft Part 1 now. Each post should reference "Part X of 5" and build on the previous one. Schedule Part 1 for [date] at 10am.
```

### Event Promotion

```
I'm hosting a [event type] on [date] about [topic]. Create a LinkedIn post that:
- Announces the event
- Lists 3 things attendees will learn
- Includes date, time, and location/link
- Has a clear CTA to register
- Adds relevant hashtags

Schedule it for [date] at 9am, and create a reminder post for the day before the event.
```

### Celebrate a Win

```
We just [achievement]. Write a LinkedIn post that:
- Celebrates the milestone without being braggy
- Thanks the team
- Shares one lesson learned along the way
- Ends with what's next

Keep it authentic, not corporate. Post it now.
```

## Tips for Better Results

1. **Draft first, post second.** Always ask Claude to show you the draft before posting. Say "show me the draft first" or "don't post yet."

2. **Be specific about tone.** "Professional but conversational", "excited but not hype-y", "thoughtful and reflective" -- Claude adapts.

3. **Provide context.** The more background you give, the better the post. Share your role, audience, and what you want readers to do.

4. **Use scheduling for consistency.** Batch-create content on Sunday, schedule for the week. Consistency beats volume on LinkedIn.

5. **Check your queue.** Periodically say "list my scheduled posts" to make sure everything is on track.

6. **Time zones matter.** Schedule using your local time and let Claude convert, or use explicit ISO 8601 with timezone offset.
